Customkm Menu plugin adds custom menus, submenus, fields, shortcode, and post types to your WordPress site, enhancing its functionality.

== Description ==

Customkm Menu plugin allows you to:

- Add custom menus and submenus to your WordPress site.
- Create custom fields to collect additional information.
- Integrate custom post types to organize your content efficiently.
- Utilize shortcodes for embedding content into posts and pages.

== Frequently Asked Questions ==

1. How do I add a custom menu?
   To add a custom menu, navigate to the Customkm Menu settings page in the WordPress admin dashboard. From there, you can easily create and manage custom menus and submenus.

2. Can I use shortcodes with this plugin?
   Yes, Customkm Menu plugin provides shortcodes that allow you to embed custom content into your posts and pages. Simply use the provided shortcode with the appropriate parameters.
